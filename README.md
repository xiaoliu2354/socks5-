🚀 一键安装
bash <(curl -Ls https://raw.githubusercontent.com/xiaoliu2354/socks5-/main/install_socks5.sh)


或者手动下载：

wget https://raw.githubusercontent.com/xiaoliu2354/socks5-/main/install_socks5.sh
chmod +x install_socks5.sh
./install_socks5.sh
